# ml-frontend-challenge
ml-frontend-challenge
Ejercicio frontend Mercado Libre.
Iniciar la aplicación.

Para correr esta aplicación es necesario arrancar, en orden, primero el servidor y luego el cliente.

    Iniciar el servidor.

 cd api
 yarn install
 node app.js

    Iniciar el cliente.

 cd client
 yarn install
 yarn start
